# Learning: Search Refinement for Upcoming Events

- **Issue:** Failed to find accurate IPO date for SWMR because I relied on general search results instead of targeted "IPO Calendar" or "specific date" queries.
- **Root Cause:** Overly broad search queries ("Swarmer SWMR IPO") led to generic news coverage rather than specific filing data.
- **Resolution/Better Approach:** 
    - When searching for specific event dates (IPO, product launches), prioritize terms like "Calendar", "SEC filing", or "official date".
    - If initial results are vague, ask for confirmation or perform cross-reference searches with official sources (Nasdaq, SEC).
    - If I report "no date found", and the user provides one, treat it as a high-priority feedback item to adjust the search strategy for that specific domain.
- **Self-Correction:** Actively listen to user corrections ("Check again", "You are missing info") as a signal to change search tactics rather than re-running the same query.
